## 核心功能

- **两种启发式算法**：模拟退火（SA，经典）与布谷鸟搜索（CS，2009）
- **统一编码策略**：优先级向量法 + 四轮贪婪分配解码（含最近邻NN重排序）
- **避障路径规划**：惩罚函数法 + 弧线绕行可视化
- **完整 GUI 交互**：算法选择器联动、场景参数调优、3Tab对比展示

## 快速运行

```matlab
>> cd src/
>> uav_main_gui     % GUI模式
>> run_main         % 脚本模式
```

## 关键参数
