# 异构无人集群海事巡检任务分配仿真系统 — 运行说明

## 课设题目五：模拟退火（SA）vs 布谷鸟搜索（CS）

---

## 如何运行

将 `src/` 文件夹（共 **16 个** `.m` 文件）复制到任意位置，在 MATLAB 中打开 `src/` 目录。

### 方式一：GUI 界面（推荐演示用）

```matlab
>> uav_main_gui
```

界面左侧为参数面板，右侧为可视化结果。按顺序操作：

| 步骤 | 操作 | 说明 |
| :--- | :--- | :--- |
| 1 | 顶部选择算法 **SA** 或 **CS** | 面板自动切换为对应算法参数 |
| 2 | 设置场景参数 → 点击 **「生成场景」** | 随机生成全动态目标、障碍物、UAV |
| 3 | 检查算法参数（左侧自动显示当前算法参数） | |
| 4 | 点击 **「运行当前算法」** | 运行优化 + 自动动态仿真 → 显示实时轨迹 |
| 5 | 切换顶部 Tab 查看结果 | 路径图 / 收敛曲线 / 分配表 |
| 6 | 切换算法再点「运行当前算法」 | 运行另一种算法，结果自动对比 |

> **核心特性：** 运行后路径图中 **实线 = UAV 实时跟踪移动目标的实际轨迹**，虚线 = 规划参考路径。菱形目标带速度箭头表示运动方向。

右侧顶部三个 Tab 可切换视图：
- **🗺 路径图（实时轨迹）** — 核心视图：实线轨迹 + 虚线规划 + 动态目标/障碍物
- **📈 收敛曲线** — SA vs CS 收敛过程对比
- **📋 分配表** — 每架无人机的目标序列和利用率

### 方式二：脚本运行（批量实验用）

```matlab
>> run_main
```

自动生成场景 → 跑 SA → 跑 CS → 动态仿真 → 打印结果对比 → 可视化。

改参数直接在 `run_main.m` 里修改。

## 参数说明

### ❶ 算法选择（GUI 顶部）

| 控件 | 说明 |
| :--- | :--- |
| **下拉框**（SA / CS） | 切换时左侧参数面板自动切换显示对应算法的参数 |

### ❷ 场景参数

| 参数 | 默认值 | 作用 | 调大效果 |
| :--- | :--- | :--- | :--- |
| **无人机 N** | 6 | 无人机总数 | 每架分到的目标变少 |
| **目标数 M** | 10 | 全部为动态目标 | 问题更复杂 |
| **障碍数 K** | 6 | 全部为动态障碍物 | 路径绕行增多 |
| **最大航程** | 200 | 各UAV的航程上限 | 可访问更多目标 |
| **目标速度** | 1.0~3.0 | 目标移动速度范围 | 目标运动更快，追踪更难 |
| **障碍速度** | 0.3~1.0 | 障碍物移动速度范围 | 障碍变化更快，碰撞风险增加 |

### ❷ 无人机异构参数

| 参数 | 默认值 | 说明 |
| :--- | :--- | :--- |
| **各UAV速度** | 8,10,6,8,7 | 逗号分隔，长度 = N |
| **传感器类型** | 1,2,3,1,2 | 1=光学 2=雷达 3=多光谱 |
| **传感器匹配** | — | 目标的所需传感器 ≤ UAV携带传感器时才能分配 |

### ❸ SA 参数（选SA时显示）

| 参数 | 默认值 | 作用 | 经验值 |
| :--- | :--- | :--- | :--- |
| **T0** | 100 | 初始温度（越大→初期越容易接受差解） | 50~200 |
| **α** | 0.95 | 降温速率（越接近1→降温越慢→搜索越充分） | 0.90~0.99 |
| **T_end** | 1 | 终止温度 | 0.1~10 |
| **最大迭代** | 400 | 迭代次数上限 | 200~1000 |

### ❹ CS 参数（选CS时显示）

| 参数 | 默认值 | 作用 | 经验值 |
| :--- | :--- | :--- | :--- |
| **Npop** | 25 | 种群规模（越大→多样性越好→越慢） | 15~50 |
| **Pa** | 0.25 | 发现概率（越大→丢弃劣解越多→开发增强） | 0.1~0.5 |
| **α_step** | 0.01 | 莱维飞行步长（越大→探索范围越大） | 0.001~0.05 |
| **最大迭代** | 400 | 迭代次数上限 | 200~1000 |

### ❺ 权重 & 仿真参数

| 参数 | 默认值 | 作用 |
| :--- | :--- | :--- |
| **ω** | 0.3 | 航程权衡系数。ω=0→只看收益；ω越大→越省航程 |
| **P** | 50 | 障碍穿越惩罚常数。越大→越不敢穿障碍 |
| **安全距离** | 2 | 障碍物安全裕量（半径增加值） |
| **仿真步数** | 300 | 动态仿真的步数上限（运行后自动执行动态仿真） |
| **随机种子** | 空 | 填数字可重现场景（如42） |

## 算法对比要点（设计报告分析方向）

| 对比维度 | SA 特点 | CS 特点 |
| :--- | :--- | :--- |
| 搜索方式 | 单点串行搜索 | 群体并行搜索 |
| 跳出局部最优 | Metropolis 准则（概率接受差解） | 莱维飞行（长步跳跃）+ 发现丢弃 |
| 收敛速度 | 依赖降温速率，中前期快 | 种群大时慢，但更稳定 |
| 参数敏感性 | 对 α 敏感 | 对 Pa 和 α_step 敏感 |
| 动态适应性 | 单点不易适应大变化 | 群体多样性更适合动态环境 |

## MATLAB 版本要求

- 兼容 MATLAB R2020b 及以上版本

- 程序化 GUI（不依赖 App Designer / GUIDE）

## 文件清单（共 15 个 .m 文件）

```
src/
├── uav_main_gui.m              # GUI（算法选择器 + 动态面板 + 3Tab）
├── run_main.m                  # 脚本入口（全动态场景）
├── generate_scenario.m         # 全动态场景生成（所有目标/障碍可移动）
├── decode_individual.m         # 优先级向量 → 分配方案（含传感器匹配）
├── calc_objective.m            # 目标函数 J = ΣR - ω·Σ(L/Lmax)
├── distance_penalty.m          # 惩罚距离
├── collision_check.m           # 线段-障碍碰撞检测
├── simulated_annealing.m       # SA 算法（Swap/Insert/Reverse + Metropolis）
├── cuckoo_search.m             # CS 算法（Lévy飞行 + 发现丢弃）
├── levy_flight.m               # 莱维飞行（Mantegna 算法）
├── simulate_dynamic.m          # 全动态仿真引擎（实时跟踪移动目标）
├── astar_path_planning.m       # 栅格 A* 避障路径规划
├── plot_path.m                 # 路径图（实线=实时轨迹 虚线=规划路径）
├── plot_convergence.m          # 收敛曲线
├── plot_allocation_table.m     # 分配表（含异构参数信息）
└── utils/                      # 工具函数目录
```
