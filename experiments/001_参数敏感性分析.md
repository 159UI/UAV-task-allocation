# 实验 001：参数敏感性分析

## 目的
分析 SA 和 CS 算法关键参数对收敛性能和解质量的影响，确定推荐参数范围。

## 实验设计

### SA 参数分析

被测试参数（每次只变化一个，其余固定）：

| 参数 | 测试值 | 固定值 |
| :--- | :--- | :--- |
| 初始温度 T0 | 10, 50, **100**, 200, 500 | T0=100 |
| 降温速率 α | 0.80, 0.90, **0.95**, 0.98, 0.99 | α=0.95 |
| 最大迭代次数 | 100, 200, **500**, 1000 | iter=500 |

### CS 参数分析

| 参数 | 测试值 | 固定值 |
| :--- | :--- | :--- |
| 种群规模 N_pop | 10, **25**, 50, 100 | N_pop=25 |
| 发现概率 Pa | 0.10, 0.15, 0.20, **0.25**, 0.35, 0.50 | Pa=0.25 |
| 步长缩放因子 α_step | 0.001, **0.01**, 0.05, 0.1 | α_step=0.01 |

### 固定场景

```
N = 5, M = 20, K = 8, L_max = 150
omega = 0.3, P = 50, d_safe = 2
seed = 42（固定随机种子）
```

## 运行方式

```matlab
% 修改 run_main.m 中的参数后运行
run_main
```

## 初步结论

（待分析）

- T0 太小 → 陷入局部最优；太大 → 收敛慢
- α 越接近 1 → 搜索越充分但耗时
- N_pop 越大 → 多样性越好但计算量线性增长
- Pa 控制探索/开发平衡
